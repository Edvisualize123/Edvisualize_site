Bounding Box Annotation

box annotation is basically used to train autonomous vehicles to detect the various objects on the streets like lanes, traffic, potholes, signals, and other objects.  


Segmentation

Semantic segmentation annotation helps train computer vision based AI models by assigning each pixel in an image to a specific class of object


Key Point Annotation

Key Point annotation is a more detailed protocol of image annotation used to detect small objects and shape variations by marking locations of Key Points.


Machine Learning
